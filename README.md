## 【发帖格式】在 Issue 发表组队需求时参照本文

标题：【大致方向】+【名称】+【相关技术/平台】+【组队所需人数】+【联系方式】

例子：【游戏】+【村民月芽大战僵尸小游戏】+【微信小程序】+【3人】+【微信/QQ】

内容：写明本项目的最终目标，实现方式，初步想法，以及联系方式/群内 ID 等。

回复：如果有任何想法，或者想要加入，可进行回复。

不符合格式的帖子会通知大家修改一下的。
